# ⛅WeatherNow — Live Weather at Your Fingertips
Ever wondered what the weather is like on the other side of the world?
WeatherNow lets you find out in seconds! 🌍
Built with pure HTML, CSS and JavaScript — no frameworks, no fluff.
Just real-time weather data powered by the OpenWeatherMap API.

## 🚀Features

- 🔍 Search any city in the world
- 🌡️ Live temperature in Celsius
- 💧 Humidity levels
- 💨 Wind speed
- ⛅ Current weather condition
- ⌨️ Press Enter or click Search — your choice!

## 🛠️Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- OpenWeatherMap API
